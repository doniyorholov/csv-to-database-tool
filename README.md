\# CSV to Database Automation Tool



\## Overview

This project is a simple Python tool that reads data from a CSV file, performs basic data cleaning, and stores the cleaned data into a SQLite database.  

It was built to demonstrate core Python skills, data handling, and basic database integration.



\## Features

\- Reads structured data from a CSV file

\- Removes incomplete or invalid records

\- Stores data in a SQLite database

\- Uses Python libraries commonly used in industry



\## Technologies Used

\- Python

\- pandas

\- SQLite

\- Git \& GitHub
